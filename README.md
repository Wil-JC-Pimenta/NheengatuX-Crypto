# Crypto Dashboard

## NheengatuX Crypto

### Português(BR)

O **Crypto Dashboard** é um sistema web interativo que permite aos usuários explorar e acessar recursos importantes de várias blockchains, como Solana, Bitcoin, Binance Smart Chain e outros. O painel exibe informações detalhadas, incluindo exploradores de blockchain, recursos de desenvolvimento, plataformas DeFi, marketplaces de NFT e muito mais.

Este sistema foi desenvolvido utilizando **Vite** como bundler, **TypeScript** para tipagem estática e **Lovable** para componentes de interface, garantindo uma experiência rápida, segura e eficiente tanto para desenvolvedores quanto para usuários.

Além disso, o sistema divulga informações do token **NheengatuX (NGTX)**,desenvolvido na rede mainnet da Solana, através do do SPL Token Mint, permitindo que os usuários explorem e interajam com informações sobre o token.

## Funcionalidades em desenvolvimento

-**Back End**
-\*Integração de Smart Contract e Stripe Payment\*\*

- **Exploração de Blockchains**: Acesso rápido a exploradores de transações e blocos para várias redes, incluindo Solana, Bitcoin, Binance Smart Chain.
- **Plataformas DeFi**: Links para as principais plataformas DeFi (Finanças Descentralizadas) para cada blockchain, permitindo que os usuários interajam com o ecossistema.
- **NFT Ecosystem**: Exploração de plataformas e marketplaces de NFTs para cada blockchain, como Magic Eden para Solana e Binance NFT.
- **Recursos de Desenvolvimento**: Ferramentas e documentação para desenvolvedores interessados em construir sobre essas blockchains, com suporte para desenvolvimentos em várias linguagens, incluindo Solidity e Rust.
- **NheengatuX (NGTX)**: Integração com o token.

## Tecnologias Utilizadas

### Bitcoin Blockchain & Resources:

Explore a Binance Smart Chain, recursos DeFi, marketplaces de NFTs e ferramentas de desenvolvimento.

## Arquitetura

A arquitetura do sistema foi construída com foco em escalabilidade e eficiência, utilizando uma arquitetura de **SPA (Single Page Application)** que facilita a interação com o usuário de maneira rápida e fluida. A comunicação com as APIs de blockchain é feita de forma assíncrona, garantindo que as informações sejam carregadas dinamicamente.

A estrutura do projeto é organizada em componentes reutilizáveis, com o uso de hooks personalizados em React para gerenciamento de estado e requisições API. Além disso, a interface de usuário é responsiva, adaptando-se a diferentes dispositivos e tamanhos de tela utilizando **Tailwind CSS**.

### Componentes principais da arquitetura:

- **Frontend (React + TypeScript)**: Responsável pela interação com o usuário e exibição das informações de blockchain, DeFi, NFTs, NGTX, etc.
- **API Gateway**: Um único ponto de entrada para todas as requisições feitas ao servidor backend (quando necessário), facilitando o roteamento de dados entre as APIs de blockchain e a interface do usuário.
- **APIs de Blockchain**: Interfaces para acessar dados das redes Solana, Bitcoin, Binance Smart Chain, NGTX e outros.
- **Requisições Assíncronas**: Todas as interações com as APIs de blockchain são feitas via chamadas assíncronas para garantir uma experiência fluida.
- **Design Responsivo**: Utiliza Tailwind CSS para garantir uma experiência otimizada em dispositivos móveis e desktops.

## English

# Crypto Dashboard

## NheengatuX Crypto

### Description

The **Crypto Dashboard** is an interactive web system that allows users to explore and access important resources from various blockchains, such as Solana, Bitcoin, Binance Smart Chain, and others. The dashboard displays detailed information, including blockchain explorers, development resources, DeFi platforms, NFT marketplaces, and much more.

This system was developed using **Vite** as a bundler, **TypeScript** for static typing, and **Lovable** for UI components, ensuring a fast, secure, and efficient experience for both developers and users.

Additionally, the system provides information about the **NheengatuX (NGTX)** token, developed on the Solana mainnet through the SPL Token Mint, allowing users to explore and interact with data about the token.

## Features in Development

- **Back End**
  - _Smart Contract and Stripe Payment Integration_
- **Blockchain Exploration**: Quick access to transaction and block explorers for multiple networks, including Solana, Bitcoin, Binance Smart Chain.
- **DeFi Platforms**: Links to the top DeFi (Decentralized Finance) platforms for each blockchain, enabling users to interact with the ecosystem.
- **NFT Ecosystem**: Explore platforms and NFT marketplaces for each blockchain, such as Magic Eden for Solana and Binance NFT.
- **Development Resources**: Tools and documentation for developers interested in building on these blockchains, with support for development in various languages, including Solidity and Rust.
- **NheengatuX (NGTX)**: Integration with the token.

## Technologies Used

### Frontend:

- **Vite**: High-performance bundler for frontend development.
- **TypeScript**: Static typing that improves code reliability and maintenance.
- **Lovable**: Framework/lib for simple, ready-to-use UI components, providing an agile and modern development experience.
- **React.js**: JavaScript library for building dynamic user interfaces.
- **Tailwind CSS**: Utility-first CSS framework for responsive, modern design.
- **Lucide Icons**: Icon package for UI.

### Blockchain APIs:

- Public APIs for **Solana**, **Binance Smart Chain**, **Bitcoin**, **NheengatuX (NGTX)**, and others to access transaction data and block information.

### Solana Network & Resources:

Explore the Solana blockchain, view transactions, use development tools, and interact with the NFT ecosystem.

### Bitcoin Blockchain & Resources:

Explore transactions on the Bitcoin network, learn how to interact with the blockchain, and access useful resources.

### Binance Smart Chain & Resources:

Explore Binance Smart Chain, DeFi resources, NFT marketplaces, and development tools.

### Links to DeFi and NFT Marketplaces:

Access the best DeFi resources and NFT marketplaces for each blockchain, facilitating interaction with decentralized finance platforms and digital assets.

### NheengatuX (NGTX) Token:

Integration with the **NheengatuX (NGTX)** token allows users to explore information about this token and interact with the ecosystem, including tracking transactions and price history.

## Architecture

The system architecture was designed with scalability and efficiency in mind, using a **SPA (Single Page Application)** architecture that facilitates fast and smooth user interaction. Communication with blockchain APIs is asynchronous, ensuring that information is dynamically loaded.

The project structure is organized into reusable components, using custom React hooks for state management and API requests. Additionally, the user interface is responsive, adapting to different devices and screen sizes using **Tailwind CSS**.

### Main Architecture Components:

- **Frontend (React + TypeScript)**: Responsible for user interaction and displaying blockchain, DeFi, NFTs, NGTX, etc., information.
- **API Gateway**: A single entry point for all requests to the backend server (when needed), facilitating data routing between blockchain APIs and the user interface.
- **Blockchain APIs**: Interfaces for accessing data from Solana, Bitcoin, Binance Smart Chain, NGTX, and others.
- **Asynchronous Requests**: All interactions with blockchain APIs are made via asynchronous calls to ensure a smooth experience.
- **Responsive Design**: Uses Tailwind CSS to ensure an optimized experience on mobile devices and desktops.
